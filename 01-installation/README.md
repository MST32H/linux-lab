
# Ubuntu Server Installation

## Objective

Install Ubuntu Server and prepare it for Linux administration.

---

## Requirements

- Ubuntu Server 24.04 LTS
- VMware Workstation
- 2 CPU
- 4 GB RAM
- 30 GB Disk

---

## Commands

```bash
sudo apt update
sudo apt upgrade -y
hostnamectl
ip a
```

---

## Verification

```bash
hostnamectl
ip a
lsb_release -a
```

---

## Result

Ubuntu Server installed successfully.
